Steam 特惠提醒插件

一个 AstrBot 专用 的 Steam 优惠查询与降价提醒插件，支持：

• 查询当前 Steam 特惠（最多 30 条）
• 按游戏单独订阅折扣提醒
• 达到阈值后自动推送到当前会话（Telegram / 微信）

───

功能说明

A) 查看特惠

返回按折扣排序的特惠列表（默认最多 30 条）。

每条包含：

• 游戏名
• 折扣百分比
• 折后现价
• 原价

B) 订阅单个游戏降价提醒

你可以给某个游戏设置折扣阈值（例如 20%）。
当该游戏折扣达到或超过阈值时，插件会主动提醒。

C) 自动提醒规则

• 后台定时轮询（默认每 300 秒）
• 折扣达到阈值会提醒
• 同一折扣不会重复狂刷
• 折扣降到阈值以下后，再次升上来会再次提醒
• 订阅/更新阈值后会立刻检查一次（若已满足阈值可即时提醒）

───

使用方法

1) 指令方式（推荐）

查看特惠

/steam特惠

订阅游戏

/steam订阅 游戏名 [折扣阈值]

示例：

/steam订阅 黑神话 悟空 20
/steam订阅 幻兽帕鲁 25

不填阈值时默认 1（即有折扣就提醒）

查看我的订阅

/steam我的

取消订阅

/steam取消 游戏名或appid

示例：

/steam取消 黑神话
/steam取消 1623730

───

2) 消息事件触发（不加斜杠也可）

支持自然输入：

• steam特惠
• 订阅steam 黑神话 悟空 20
• 取消steam 黑神话
• 我的steam

───

配置项

配置文件：

data/config/astrbot_plugin_steam_deal_alert_config.json

• cc：价格地区（默认 cn）
• lang：语言（默认 schinese）
• poll_seconds：轮询间隔秒数（默认 300）
• top_deals_limit：特惠条数上限（默认 30）

───

说明

• 数据来源为 Steam 官方接口/页面结果
• 插件不会上传订阅数据到第三方
• 本地订阅数据存放在：

data/plugin_data/astrbot_plugin_steam_deal_alert/subscriptions.json

───

常见问题

Q1：我订阅了但没提醒？

先用 /steam我的 确认订阅是否成功落库。

Q2：为什么有时名字显示异常？

插件会优先使用 Steam 返回名；若异常，会回退到你订阅时记录的游戏名。
